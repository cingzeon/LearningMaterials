## 依赖技术点
   - webpack
        $ npm init -y                               // 初始化
        $ mkdir src                                 // 创建文件夹
        $ touch index.html README.mkd               // 创建文件
        $ npm install webpack@1.13.2 -g             // 全局环境的安装 
        $ npm install webpack@1.13.2 --save-dev     // 本地环境的安装
        $ touch webpack.config.js                   // 创建一个webpack.config.js 的配置文件
        


   - babel 语法解析
   - es6/7 基本语法
   - eslint 语法检查
   - npm script 统一任务构建
   - react 组件 基础类库
   - mocha
   - karma 单元测试
